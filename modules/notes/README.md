# Notes Module

Personal, read-only knowledge module.

## Purpose
This module provides a structured space for personal notes,
research, ideas, and references.

It is:
- read-only by design
- human-maintained
- non-executable

## Structure
- topics/ — thematic notes
- references/ — sources and links

## Governance
- Authority Level: ADVISORY
- Execution: FORBIDDEN
- Mutation: MANUAL ONLY

This module is described and reviewed via Sapianta Chat.
