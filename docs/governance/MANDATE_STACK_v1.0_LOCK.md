Pomen zaklepa

Z zaklepom MANDATE STACK v1.0 velja:

mandatni model je dokončen

spremembe niso dovoljene brez nove fazne odločitve

implementacija mora biti 100 % skladna

noben del sklopa ne sme biti:

reinterpretiran,

razširjen,

implicitno obit,

UX-ovsko zmehčan

Kanonična načela (zavezujoča)

Zaustavitev je vedno mogoča

Audit je pasiven

Vidnost je read-only

Posledice za implementacijo

Vsaka implementacija, ki:

omogoča real-time kontrolo prek UI,

združuje vidnost in vpliv,

implicitno ustvarja ali spreminja mandate,

razlaga varnostne ali governance razloge,

je nezdružljiva z zaklenjenim mandatnim skladom.
